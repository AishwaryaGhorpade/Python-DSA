What is Inheritance in Python?

Inheritance allows one class (child) to acquire the properties and methods of another class (parent).

In simple words:

A child class can use and modify everything from the parent class without rewriting the code.

This helps with:

Code reusability

Clean structure

Reducing duplication

Extensibility

â­ 1. Single Inheritance

One parent â†’ One child.

class A:
    def show_A(self):
        print("Class A method")

    def show_B(self):
        print("Class B method")

obj = B()
obj.show_A()
obj.show_B()

â­ 2. Multilevel Inheritance

Parent â†’ Child â†’ Grandchild.

class A:
    def show_A(self):
        print("A")

class B(A):
    def show_B(self):
        print("B")

class C(B):
    def show_C(self):
        print("C")

obj = C()
obj.show_A()
obj.show_B()
obj.show_C()

â­ 3. Multiple Inheritance

Child inherits from multiple parents.

class A:
    def method_A(self):
        print("From A")

class B:
    def method_B(self):
        print("From B")

class C(A, B):
    pass

obj = C()
obj.method_A()
obj.method_B()

â­ 4. Hierarchical Inheritance

One parent â†’ multiple child classes.

class Parent:
    def parent_method(self):
        print("Parent")

class Child1(Parent):
    def child1_method(self):
        print("Child1")

class Child2(Parent):
    def child2_method(self):
        print("Child2")

obj1 = Child1()
obj2 = Child2()

obj1.parent_method()
obj2.parent_method()

â­ 5. Hybrid Inheritance

Combination of multiple inheritance types.

Example: Hierarchical + Multiple.

â­ 6. Using super() in Inheritance

super() is used to call parent class methods.

class Parent:
    def show(self):
        print("Parent show")

class Child(Parent):
    def show(self):
        super().show()
        print("Child show")

obj = Child()
obj.show()


Output:

Parent show
Child show

ğŸ”¥ Constructor Inheritance

ğŸ”¥ Overriding Constructor

Child defines its own constructor.

class Parent:
    def __init__(self):
        print("Parent constructor")

class Child(Parent):
    def __init__(self):
        print("Child constructor")

obj = Child()


Output:

Child constructor


Parent constructor is not called unless you manually call:

super().__init__()
