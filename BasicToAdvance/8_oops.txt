ðŸ§© 2. Abstraction

Abstraction means hiding complex implementation and showing only the required things.

Python has abstract classes using the abc module.

Example:
from abc import ABC, abstractmethod

class Animal(ABC):

    @abstractmethod
    def sound(self):
        pass   # only declaration â†’ no implementation

class Dog(Animal):
    def sound(self):
        return "Bark"


Why abstraction?

User sees what the method does, not how it works.

Encourages clean, maintainable code.

Even normal methods hide logic, but abstract methods force child classes to implement the method â€” thatâ€™s the difference.

ðŸ§© 4. Polymorphism

Polymorphism = one name, many forms.

Method overriding Example:
